# GEO Audit — definition of done

The report at `/mnt/session/outputs/geo-audit-report.md` is satisfactory only if ALL of the following hold:

1. **Six dimensions scored.** The report scores every one of the six GEO dimensions — (A) crawl access & freshness, (B) extractability, (C) structured data, (D) entity & authority, (E) off-site citation footprint, (F) original data & content quality — each item marked OK / weak / missing.
2. **Evidence-cited.** Every finding cites specific, real evidence from the audited site (an actual URL, HTML snippet, schema block, robots.txt line, or search result) — not generic claims.
3. **AI-visibility baseline.** The report includes a baseline for every target question, stating for each whether the brand currently surfaces in AI/search answers and who is recommended instead.
4. **Prioritized two-track roadmap.** Recommendations are split into quick wins (on-site) and strategic investments (off-site & content), ordered by impact, with off-site / original-data leverage led first.
5. **No fabrication.** Every number or claim is either verifiable from observed evidence or explicitly tagged "unverified". No invented data, citations, or competitor facts.
